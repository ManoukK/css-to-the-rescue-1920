# reflectie Voor deze opdracht had ik het liefst veel meer tijd besteed aan het ontdekken van CSS. Ik vond het echt een hele leuke opdrachten ik heb al heel veel geleerd. Misschien had ik meer "gekke" dingen moeten doen. Dat mijn concept nog iets extremer is. ### SVG Een van de eisen die ik had gekozen was veel doen met svg. Naar mijn idee heb ik dit niet erg veel gedaan. Ik probeerde het wel maar ik vond svg's en svg animaties erg lastig. Ik snap nu wel beter hoe svg's werken maar wil er toch nog dieper op in duiken omdat je er veel meer mee kan dan dat je zou denken. ### Media query Mijn tweede eis was voor mij aan het begin nig niet helemaal helder. Ik wist nog niet zo goed wat ik wilde kiezen. Later in mijn proces wilde ik gebruik maken van flexbox en toen wilde ik wel de uitdaging aangaan om mijn site responsive te maken zonder media queries. Ik had aan het begin wat moeite met het begrijpen van flexbox maar toen het eenmaal werkte begreep ik waarom het zo werke. Ik vond het leuk en interessant om met flexbox te spelen. Vooral omdat school bij blok web aangaf dat je juist beter geen flexbox kan gebruiken. Ik begrijp niet zo goed waarom want het is eigenlijk best handig! ### Darkmode Deze eis was ik in de loop van mijn proces een beetje vergeten. Dit heb ik dus op het laatst nog toegevoegd. Ik wist nooit hoe het werkte maar uiteindelijk is het best een makkelijk truckje. (vooral als je al gebruik maakt van :root). Mijn idee was eerst om van saai naar kleur te gaan. Allen dit was niet hoe darkmode werkte. Je kan wel de kleuren omdraaien dus van wit kan je zwart maken. Alleen je kan niet element 1 die eerst wit was blauw maken en element 2 die eerst ook wit was roze maken. Het kan. Alleen word je root heel omslachtig en niet meer leesbaar voor anderen. Dus heb ik maar besloten om zwart en wit om te draaien en de kleuren palette die ik nu gebruik wat donkerder te maken.
